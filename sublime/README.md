# My Sublime Text 3 configuration

## Packages

Below is the list of plug-ins I like for my dev environment. I use [Package Control](https://packagecontrol.io/) to install them.

* [Docblockr](https://github.com/spadgos/sublime-jsdocs)
* [EditorConfig](https://github.com/sindresorhus/editorconfig-sublime)
* [Git Gutter](https://github.com/jisaacks/GitGutter)
* [JSHint Gutter](https://github.com/victorporof/Sublime-JSHint)
* [Pretty JSON](https://github.com/dzhibas/SublimePrettyJson)
* [Select quoted](https://github.com/int3h/SublimeSelectQuoted)
* [PHP unit completions](https://github.com/tkowalewski/phpunit-sublime-completions)
* [SideBarEnhancements](https://github.com/titoBouzout/SideBarEnhancements)
